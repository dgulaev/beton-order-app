# План: Тёмная / светлая тема мобильной админки

## Статус: Отложено (обсуждено, не первоочередное)

## Цель
Добавить переключатель тёмная ↔ светлая тема в мобильную админку (`/mobile`).
Тёмная тема — текущая (цвета `#162032`, `#25334A`, `#334155` и т.д.).
Светлая тема — белый фон, тёмный текст, контрастные акценты.

## Почему отложено
- Все цвета сейчас жёстко прописаны как inline-стили в ~15 файлах (несколько сотен значений).
- Светлая тема требует не просто инверсии — нужно отдельно проработать контраст карточек, теней, бейджей, иконок.
- Оценка трудозатрат: ~14–18 часов.
- Текущий приоритет — функциональные фичи, тема откладывается до стабилизации интерфейса.

## Технический план реализации

### 1. CSS-переменные (`app/globals.css`)
Заменить хардкод-цвета на переменные, определить два набора:
```css
:root[data-theme="dark"] {
  --bg-page:      #162032;
  --bg-card:      #25334A;
  --bg-inner:     #334155;
  --bg-element:   #1E2D40;
  --border:       #334155;
  --text-primary: #E2E8F0;
  --text-secondary: #94A3B8;
  --text-muted:   #475569;
}

:root[data-theme="light"] {
  --bg-page:      #F1F5F9;
  --bg-card:      #FFFFFF;
  --bg-inner:     #F8FAFC;
  --bg-element:   #F1F5F9;
  --border:       #E2E8F0;
  --text-primary: #0F172A;
  --text-secondary: #475569;
  --text-muted:   #94A3B8;
}
```

### 2. ThemeContext (`app/providers/ThemeProvider.tsx`)
- Хранить выбор в `localStorage` (ключ `mobile_theme`)
- Применять `document.documentElement.setAttribute('data-theme', theme)`
- Экспортировать `useTheme()` хук

### 3. Переключатель в интерфейсе
- Иконка `Sun` / `Moon` (lucide-react) в правом углу навбара `layout.tsx`
- Рядом с broadcast-индикатором или в отдельном месте

### 4. Миграция компонентов (все файлы мобильной админки)
Заменить `background: '#162032'` → `background: 'var(--bg-page)'` и т.д.:
- `app/mobile/layout.tsx`
- `app/mobile/page.tsx`
- `app/mobile/zayavki/page.tsx`
- `app/mobile/mixers/page.tsx`
- `app/mobile/warehouse/page.tsx`
- `app/mobile/clients/page.tsx`
- `app/mobile/components/MobileCalendar.tsx`
- `app/mobile/components/MobileExitButton.tsx`
- `app/mobile/driver/components/DriverDashboard.tsx`
- `app/mobile/driver/components/DriverTripDetailModal.tsx`
- `app/mobile/mixers/driver-view/[id]/page.tsx`

### 5. Проверка и доработка
- Проверить контраст всех акцентных цветов (`#10B981`, `#3B82F6`, `#FACC15`, `#EF4444`) на светлом фоне
- Проверить тени, backdrop-filter, полупрозрачные фоны
- Тест на реальных устройствах (Samsung, iPhone)
